Image search abstraction layer

FreeCodeCamp's backend challenge

Example usage:

Append your search query to the URL of the app + /api/imagesearch/ like this: https://short-url-inka.glitch.me/short/www.freecodecamp.org

Example output:

If your input is a valid URL, the output will look something like this:

{"longURL":"http://www.freecodecamp.org","shortURL":"5","_id":"5a537d7b293c374536a1e27c"}


When you recive your shortURL you can start to use it by appending it to the app URL like this: https://short-url-inka.glitch.me/5

and you will be redirected to your long URL.
