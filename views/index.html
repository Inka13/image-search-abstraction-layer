Image search abstraction layer

FreeCodeCamp's backend challenge

Example usage:

Append your search query to the URL of the app + /api/imagesearch/ like this: https://image-search-inka.glitch.me/api/imagesearch/cute%20kittens?offset=3

Where cute%20kittens is our search string and %20 is representing a blank space, and offset is the number of result page.

There are 10 results per page, so offset=3 would take you to results 30 through 39. You can ommit this parameter, and just view the results starting from 0 to 9.

You can also get 10 latest search queries like this: https://image-search-inka.glitch.me/api/latest/imagesearch
