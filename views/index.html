Image search abstraction layer

FreeCodeCamp's backend challenge

Example usage:

Append your search query to the URL of the app + /api/imagesearch/ like this: https://image-search-inka.glitch.me/api/imagesearch/cute%20kittens?offset=3 Where cute%20kittens is our search string and %20 is representing a blank space

Example output:

If there are any results, the output will look something like this:

{"longURL":"http://www.freecodecamp.org","shortURL":"5","_id":"5a537d7b293c374536a1e27c"}


When you recive your shortURL you can append it to the app URL like this: https://short-url-inka.glitch.me/5

and you will be redirected to your long URL. Warning: This is for presentation only, there is no guarantee of how long will the short links work in the future, so use them for immediate use only
